- [ ] **Step 4: Split config structs**

In `doctest/runner.h`, change the config from one struct to two:

Framework-agnostic config (in `namespace etst`):
```cpp
namespace etst {
    struct Config {
        bool (*board_init)(Print& log) = nullptr;
        void (*after_cycle)() = nullptr;
        uint32_t ready_timeout_ms = 0;
    };
    inline Config config;
}
```

Doctest-specific config (in `namespace etst::doctest`):
```cpp
namespace etst::doctest {
    struct Config {
        void (*configure)(::doctest::Context& ctx) = nullptr;
    };
    inline Config config;
}
```

Update all references from `config.board_init` to `etst::config.board_init` and `config.configure_context` to `etst::doctest::config.configure`.

- [ ] **Step 5: Remove weak function hooks**

In `doctest/runner.h`, remove the `ptr_board_init`, `ptr_configure_context`, `ptr_after_cycle` weak function declarations and their usage. Replace with config struct reads:

```cpp

}
```
